Instructions

  • Load at ½ span:
    • we can Add weights to the Beam by clicking on ADD WEIGHT button.
    • Now add the deflection readings to the observation table by clicking ADD READINGS TO TABLE button.
    • Do the above steps for { 0.5 kg, 1 kg, 1.5 kg, 2 kg} weights.
  • Remove all the load by clicking REMOVE WEIGHT button 4 times.
  • Use SWAP POSITIONS button to change the position of load from ½ span to ¼ span and deflectometer from ¼ span to ½ span.
  • load at ¼ span:
    • we can Add weights to the Beam by clicking on ADD WEIGHT button.
    • Now add the deflection readings to the observation table by clicking ADD READINGS TO TABLE button.
    • Do the above steps for { 0.5 kg, 1 kg, 1.5 kg, 2 kg} weights.
  • Calculations:
    • After adding all the recordings, now click the NEXT button.
    • Enter your calculated values of p1s12 and p2s21 in respective rows to check answers by clicking VERIFY button..
    • Check your answers and click SHOW RESULT button to view the answer.
  • you can use SHOW DATA button whenever you want to view the data.
  • Use PRINT OBSERVATIONS button to save the experiment.
