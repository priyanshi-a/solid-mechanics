Instructions

  • Loading phase:
    • we can Add weights to the Beam by clicking on ADD WEIGHT button.
    • Now add the deflection readings to the observation table by clicking ADD READINGS TO TABLE button.
    • Do the above steps for { 0.5 kg, 1 kg, 1.5 kg, 2 kg, 2.5kg} weights.
  • Unloading phase:
    • Remove weights from the stand by clicking REMOVE WEIGHT button.
    • Now add the deflection readings to the observation table by clicking ADD READINGS TO TABLE button.
    • Do the above steps for { 2 kg, 1.5 kg, 1 kg, 0.5 kg, 0kg} weights.
  • Calculations:
    • After adding all the recordings, now click the CALCULATE button.
    • Use the formulas to calculate the young’s modulus and click NEXT button.
    • Enter your calculated values in respective rows to check answers by clicking VERIFY button.
    • Check your answers and click SHOW RESULT button to view the answer.
  • you can use SHOW DATA button whenever you want to view the data.
  • Use PRINT OBSERVATIONS button to save the experiment.
  • Use RESET button to start the experiment from beginning.
