Instructions

  • Click NEXT button below the setup to arrange the rod holder equipment.
  • Now again click NEXT button to insert the specimen into rod holder.
  • Click the NEXT button once more to place the setup in Universal testing machine.
  • Click the START button to start the machine.
  • Click the STOP button if you want to stop the machine.
  • Once the rod is broken, load readings will be added to the observation table automatically.
  • Now calculate the Ultimate Shear Stress from the data in observation table.
  • Now click NEXT button and enter the calculated Ultimate Shear Stress value.
  • Press the VERIFY button to check the correctness.
  • Click the SHOW RESULT button to view result.
  • you can use SHOW DATA button whenever you want to view the data.
  • Use PRINT OBSERVATIONS button to save the experiment.
  • Use RESET button to start the experiment from beginning.
