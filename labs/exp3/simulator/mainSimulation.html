Instructions

  • Click LOAD button to start loading the spring for expansion, values are automatically added to the observation table.
  • Click UNLOAD button to remove load from spring for compression, values are automatically added to the observation table.
  • Calculations:
    • After adding all the recordings, now click the NEXT button.
    • Use the formula to calculate the rigidity modulus.
    • Enter your calculated values in respective rows to check answers by clicking VERIFY button.
    • Check your answers and click SHOW RESULT button to view the answer.
  • you can use SHOW DATA button whenever you want to view the data.
  • Use PRINT OBSERVATIONS button to save the experiment.
  • Use RESET button to start the experiment from beginning.
